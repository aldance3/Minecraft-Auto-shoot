This is an auto-shoot script for the minecraft server statesmc.us (specifically designed for the industrial rifle gun)
Requirements:
  Zergatul Cheatutils Version 26.1.x - https://github.com/Zergatul/cheatutils
  Advanced scripting option on in cheatutils settings
  Placed in http://localhost:5005/#/key-bindings (NOT events scripting)
Extras:
  Meteor client
  Meteorist - Meteor client addon (provides aimbot provided you use the correct settings,)
Please read the license before use.
